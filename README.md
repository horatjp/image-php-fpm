# PHP FPM Docker Image

This image is based on the official PHP FPM image and adds some additional PHP extensions.

## PHP Extensions

- bcmath
- exif
- gd (with freetype and jpeg support)
- imap
- imagick
- intl
- mbstring
- mysqli
- opcache
- pcntl
- PDO (MySQL, PostgreSQL, SQLite)
- redis
- xml
- xdebug
- zip
